1 - Feature-Packs

Hier geht es zur vollständigen, englischen Dokumentation.

1.1 - Basis-Features

Erstellen Sie Verknüpfungen zu Ihren Dateien und Ordnern.

Erstellen Sie Verknüpfungen zu Ihren Dateien und Ordnern, speichern Sie diese an einem beliebigen Ort und öffnen Sie die Verknüpfungen automatisch. Diese App ersetzt „file:“-Links und ist für alle Ihre iPhones, iPads und Macs geeignet.

1.2 - Arbeiten mit Links

Für optimierte Workflows beim Verlinken.

Das Funktionspaket „Links“ ermöglicht optimierte Workflows beim Verlinken.

Kürzlich geöffnete Links wieder aufrufen

Behalten Sie den Überblick über Links, die Sie aus anderen Apps geöffnet haben. Sie können diese später einfach wieder öffnen und zu Ihren Notizen hinzufügen.

Links für später speichern

Beim Durchsuchen können Sie Links ganz einfach als Lesezeichen speichern, um sie später wieder zu öffnen oder zu Ihren Notizen hinzuzufügen.

Beliebige Metadaten hinzufügen

Sie können Links im Dateibrowser beliebige Metadaten hinzufügen. Der Editor speichert die verwendeten Attribute und -werte. Taggen Sie Ihre Links, fügen Sie Info-Notizen hinzu und verknüpfen Sie verwandte Links. Sehen Sie, wo der aktuelle Link in anderen Metadaten erwähnt wird.

Links in Notizen sammeln

In Notizen können Sie einfachen Text, Links und einfache Auszeichnungssprachen beliebig kombinieren. Nutzen Sie Notizen, um eine geordnete Liste von Links zu erstellen oder eine Geschichte zu erzählen. Wählen Sie eine Aktuelle Notiz aus und fügen Sie ihr ganz einfach Links über das Kontextmenü hinzu.

Erweiterte Verlinkung nutzen

Kopieren Sie Ihre Links in andere Apps mit verschiedenen Titelformaten. Exportieren Sie bei Bedarf kompatible Weblinks. Durch Tippen auf diese Weblinks gelangen Sie zurück zu dieser App.

Widgets auf dem Startbildschirm verwenden

Diese App bietet Widgets, die Sie Ihrem Startbildschirm hinzufügen können, um Links direkt zu öffnen.

1.3 - Zugehörige Dateien

Arbeiten mit verwandten Dateien und Ordnern mithilfe virtueller Ordner, Overlays, Geschwisterdateien und Anmerkungsdateien.

Das Funktionspaket Verwandte Dateien ermöglicht das Arbeiten mit Dateien, die über Dateinamen oder Pfad miteinander verbunden sind.

Virtuelle Ordner-Overlays erstellen

Sie können mehrere Basisordner und Basisdateien in einem virtuellen Ordner zusammenfassen. Durch die Verwendung gleicher Unterordnernamen erhalten Sie eine Art Overlay-Dateisystem, das Sie durchsuchen und verlinken können. Wenn Sie keine Schreibrechte für einen freigegebenen Basisordner haben, erstellen Sie einfach einen beschreibbaren Basisordner und führen Sie beide Ordner zu einem virtuellen Ordner zusammen.

Ordner-Overlays an beliebiger Stelle hinzufügen

Standardmäßig werden Overlays für alle Basisordner und virtuellen Ordner bereitgestellt. Sie können problemlos parallele Ordnerstrukturen für jeden Host erstellen und dort verwandte Dateien ablegen.

Gleichnamige Dateien und Ordner anzeigen

Beim Öffnen eines Links werden andere Dateien und Ordner mit ähnlichem Namen im selben übergeordneten Ordner angezeigt. So können Sie beispielsweise Informationen zu einer Datei MyFile.ext in einem Ordner namens MyFile-Info speichern. Dies funktioniert auch für virtuelle Ordner und Standard-Overlay-Ordner. Alternativ können Sie Metadaten verwenden, wenn Sie lediglich Text hinzufügen möchten.

Anmerkungsdateien und -ordner erstellen

Beim Anzeigen einer Datei oder eines Ordners kann die App Ihnen helfen, einen gleichnamigen Ordner für Anmerkungen zu erstellen. Dies ist eine der wenigen Stellen, an denen Foldie Änderungen am zugrunde liegenden Dateisystem vornimmt.

Wenn Sie dies lieber manuell durchführen möchten, deaktivieren Sie diese Option.

Achtung: Sie fügen Inhalte zu einem beschreibbaren Ordner hinzu. Handelt es sich um einen freigegebenen Ordner, sind diese Inhalte auch für Ihr Team sichtbar. Prüfen Sie die Richtlinien Ihres Teams bezüglich der zulässigen Ergänzungen.

1.4 - Automatisierung

Kurzbefehle aufrufen und Daten für die Kurzbefehle-App freigeben.

Das Automatisierungs-Funktionspaket ermöglicht die Verwendung der Foldie-App zusammen mit der Kurzbefehle-App. Skripte mit Kurzbefehlen ermöglichen leistungsstarke Arbeitsabläufe, jedoch ist Vorsicht geboten.

Kurzbefehl-Aktionen aufrufen

Erstellen Sie benutzerdefinierte Aktionen in der Kurzbefehle-App und rufen Sie diese mit einem ausgewählten Link auf. Fügen Sie beispielsweise den ausgewählten Link einer Notiz in Apple Notizen hinzu.

Automatisieren

Verwenden Sie die Aktionen dieser App in der Kurzbefehle-App und an anderen Stellen im System.

Sicherheitshinweis

Diese Funktionen sind standardmäßig deaktiviert, um Ihre Dateien zu schützen. Aktivieren Sie diese Funktionen nur, wenn Sie genau wissen, was Sie tun. Kurzbefehle können Änderungen am zugrunde liegenden Dateisystem vornehmen.

Überprüfen Sie Kurzbefehle immer, bevor Sie sie installieren und ausführen! Sie riskieren Ihre Daten und Ihre Privatsphäre!
