# Privacy Policy

The app **Foldie Links** is a tool and data collection is kept to a minimum.

## What You can do

- Within the app the user has access to folders that he or she explicitly provided access to.
- Additionally the user can create data in the form of links, notes, metadata and settings.
- Some of the user-created data is synced accross the user's devices using iCloud to support similar workflows accross devices.


## Your data is not collected

- No user data is collected.
- No data about the usage of the apps is collected.
- No file contents from the user's file system is collected.
- No data is sent to servers other than to the private iCloud database.


## Data collection by others

- Within the app, no data is collected by 3rd parties.
- Crash reports may be collected by Apple if approved by the user.
- When leaving the app by following links to the web, navigation data may be collected on the web by 3rd parties.
