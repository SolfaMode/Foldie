# Datenschutzrichtlinie

Die App **Foldie Links** ist ein Tool, und die Datenerfassung wird auf ein Minimum beschränkt.

## Ihre Möglichkeiten

- Innerhalb der App haben Sie Zugriff auf Ordner, für die Sie explizit Zugriff gewährt haben.
- Sie können außerdem Daten in Form von Links, Notizen, Metadaten und Einstellungen erstellen.
- Einige der von Ihnen erstellten Daten werden mithilfe von iCloud geräteübergreifend synchronisiert, um ähnliche Arbeitsabläufe zu ermöglichen.

## Ihre Daten werden nicht erfasst

- Es werden keine Nutzerdaten erfasst.
- Es werden keine Daten über die Nutzung der App erfasst.
- Es werden keine Dateiinhalte aus Ihrem Dateisystem erfasst.
- Es werden keine Daten an andere Server als die private iCloud-Datenbank gesendet.

## Datenerfassung durch Dritte

- Innerhalb der App werden keine Daten von Dritten erfasst.
- Absturzberichte können von Apple erfasst werden, sofern Sie dies genehmigen.
- Wenn Sie die App verlassen und Links im Web folgen, können Navigationsdaten von Dritten im Web erfasst werden.
