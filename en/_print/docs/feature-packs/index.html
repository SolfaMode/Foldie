1 - Basic Features

Create Links to your Files and Folders.

Create Links for your files and folders, store them anywhere, and resolve your files and folders when activating the Link. This app is a replacement for „file:“-Links that is designed to work across all of your iPhones, iPads and Macs.

2 - Working with Links

For improved linking workflows.

The Links feature pack enables improved linking workflows.

Revisit recently opened Links

Keep track of links that you have opened from other apps. You can later easily re-open them and add the links to Notes.

Bookmark Links for later

While browsing, you can easily bookmark Links to re-open them later or to add them to Notes.

Add arbitrary Metadata

You can attach arbitrary Metadata to Links in the File Browser. The editor keeps track of used attribute keys and values. Tag your Links, add info-notes, attach related Links. See where the current Link is mentioned in other Metadata.

Collect Links in Notes

In notes you can freely mix plain text, links and basic markup. Use notes to create an ordered list of Links or to tell a story. Select a Current Note and easily add Links to it from context menus.

Use advanced Linking

Copy your Links to other apps using different title formats. Export compatible Web-Links where necessary. Tapping those Web-Links brings you back to this App. (Advanced links)

Use Widgets from your Home Screen

This app provides widgets that you can add to your home screen, so that you can open some links directly.

3 - Related files

Working with related files and folders using virtual folders, overlays, sibling and annotation files.

The Related Files feature pack allows you to work with files that are related by file name or path.

Construct Virtual Folder Overlays

You can arrange multiple Base Folders and Base Files into a Virtual Folder. By using the same subfolder names, you get a kind of Overlay File System, that you can browse and link to. If you do not have write access to a shared Base Folder, simply create a writable Base Folder and merge both folders into a Virtual Folder.

Add Folder Overlays anywhere

Default overlays provide overlays for all base folders and virtual folders. You can easily create parallel folder structures for each host and put related files there.

Display sibling Files and Folders

When opening a Link, other files and folders with a similar name within the same parent folder are shown. So you can put information about a file MyFile.ext in a folder called MyFile-Info. This also works across virtual folders and default overlay folders. As an alternative, use Metadata, if you just want to attach some text. (See Annotations.)

Create Annotation Files and Folders

When viewing a file or folder, the app can help you create a sibling folder for annotations. (See Annotations.) This is one of the few places, where Foldie performs changes to the underlying file system. If you prefer to do this by hand, deactivate this option.

Take care: You will add content to a writable folder. If this is a shared folder, your team will also see that content. Check your team’s policies about what is allowed to add.

4 - Automation

Call shortcuts and expose data to the Shortcuts app.

The Automation feature pack allows you to use the Foldie app together with the Shortcuts app. Scripting with Shortcuts can enable powerful workflows, but you should take care.

Call Shortcut Actions

Craft custom actions in the Shortcuts app and call those actions with a selected Link. For example, add the selected Link to a note in Apple Notes.

Automate

Use this app’s actions in the Shortcuts app and elsewhere in the system.

Security warning

These features are deactivated by default, to keep your files safe. You should only activate these features, if you know what you are doing. Shortcuts may perform changes to the underlying file system.

Always check Shortcuts before installing and running them! You risk your data and privacy!
